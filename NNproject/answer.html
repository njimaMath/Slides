1. 陰関数の微分

与えられた式 x2 + xy + y2 = 7 を x で微分する。

2x + x(dy/dx) + y + 2y(dy/dx) = 0

したがって、

(x + 2y)(dy/dx) = −(2x + y)

より、

dy/dx = −(2x + y) / (x + 2y)

を得る。

点 (1, 2) における傾きは

−(2・1 + 2) / (1 + 2・2) = −4/5

である。よって、接線の方程式は

y − 2 = −(4/5)(x − 1)

である。

2. ラグランジュ未定乗数法（最大値）

制約条件を x2 + y2 = 1 とする。ラグランジュ関数を

Φ(x, y, λ) := 3x + 4y − λ(x2 + y2 − 1)

と定義する。Φ の極値候補は各変数に関する偏微分がすべて 0 となる点だから、

Φx = 3 − 2λx = 0,   Φy = 4 − 2λy = 0,   Φλ = −(x2 + y2 − 1) = 0

を連立して解く。最初の二式から x = 3/(2λ), y = 2/λ であり、これを制約条件に代入すると

λ = ±5/2

となる。したがって極値候補は

(x, y) = (3/5, 4/5),   (−3/5, −4/5)

である。それぞれでの関数値は 5 と −5 なので、最大値を与える点は

(x, y) = (3/5, 4/5)

となる。このとき、

f(3/5, 4/5) = 3・(3/5) + 4・(4/5) = 5

したがって、最大値は 5 である。
