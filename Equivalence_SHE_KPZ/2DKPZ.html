Scheme

The simulated equation is

$ \displaystyle \partial_t h=\frac12 \Delta h+\frac12\bigl(Q(h)-C_N\bigr)+\beta\xi^\epsilon. $

The periodic grid has spacing $dx=L/N$ and uses explicit Euler time stepping with $dt=c\,dx^2$.

$ \displaystyle h_{ij}^{n+1}=h_{ij}^{n}+dt\left[\frac12\Delta_h h_{ij}^{n}+\frac12\left(Q_{ij}^{n}-C_N\right)\right]+\beta\sqrt{\frac{dt}{dx^2}}\eta_{ij}^{n}. $

The five-point Laplacian is used with periodic indexing.

$ \displaystyle \Delta_h h_{ij}=\frac{h_{i+1,j}+h_{i-1,j}+h_{i,j+1}+h_{i,j-1}-4h_{ij}}{dx^2}. $

The nonlinear term is the symmetric forward and backward gradient square.

$ \displaystyle Q_{ij}=\frac12\left[(D_x^+h)^2+(D_x^-h)^2+(D_y^+h)^2+(D_y^-h)^2\right]. $

For long browser runs, the default stabilized mode splits each displayed step into smaller internal steps and applies a smooth limiter to rare extreme values of $Q$. Turn it off to see the raw explicit Euler update.

The discrete noise represents cell-averaged space-time white noise.

$ \displaystyle \xi_{ij}^{n}=\frac{\eta_{ij}^{n}}{\sqrt{dt\,dx^2}},\qquad \mathbb E[\xi_{ij}^{n}\xi_{kl}^{m}]=\frac{\delta_{ik}\delta_{jl}\delta_{nm}}{dt\,dx^2}. $

Renormalization

The renormalization is not a change of the white noise samples. The samples still enter as

$ \displaystyle \beta\sqrt{\frac{dt}{dx^2}}\eta_{ij}^{n}. $

Wick ordering subtracts $\frac12 C_N$ from the KPZ drift, where

$ \displaystyle C_N=\frac{\beta^2(N^2-1)}{L^2}. $

When the checkbox is off, the code sets $C_N=0$, making the vertical drift visible in the true mean.

Temperature Scaling

In fixed-grid mode, $\beta$ is controlled directly. In 2D marginal KPZ scaling mode, the code uses

$ \displaystyle \beta_N=\widehat\beta\sqrt{\frac{2\pi}{\log(L/dx)}}. $

Here larger $\beta$ means lower temperature and stronger noise; $\widehat\beta>1$ is the low-temperature side.
